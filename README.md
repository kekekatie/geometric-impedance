## Silent Corruption in Aperiodic Substrates (2026)

**The exo/endo distinction:** How do aperiodic tilings (Penrose, Ammann-Beenker) respond when their relational structure is silently scrambled?

| Resource | Description |
|----------|-------------|
| [Paper (v3 matched-scale)](silent_corruption/relational_corruption_v3_matched_scale.md) | Full paper with matched-scale AB vs Penrose comparison |
| [Data & Code](silent_corruption/) | CSV results, Python scripts, figures for reproducibility |

**Key finding:** AB retains identity via external perpendicular-space coordinates (address AUC 0.986) even after graph destruction; Penrose does not (0.661). The exo/endo split measured directly.

See also: [Zenodo record](https://zenodo.org/records/15560880) for the formal publication.
